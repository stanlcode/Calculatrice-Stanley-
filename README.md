# Calculatrice Stanl

## Description

Calculatrice web simple développée en HTML, CSS et JavaScript.
Elle permet d’effectuer des opérations de base comme l’addition, la soustraction, la multiplication et la division.

---

## Fonctionnalités

* Saisie des nombres avec boutons
* Opérations : +, -, ×, /, %
* Résultat avec =
* Effacer (C)
* Supprimer un caractère (Del)
* Changer le signe (+/-)

---

## Technologies utilisées

* HTML5 : structure de la page
* CSS3 : design et mise en page
* JavaScript : logique et calcul

---

## Structure du projet

```
calculatrice-stanl/
│── index.html      # Structure de la page
│── style.css       # Design et mise en page
│── script.js       # Logique de la calculatrice
│── README.md       # Documentation du projet
```

---

## Utilisation

1. Télécharger ou cloner le projet
2. Ouvrir le fichier index.html dans un navigateur
3. Utiliser les boutons pour effectuer des calculs

---

## Remarque importante

Le projet utilise la fonction eval() pour effectuer les calculs.
Cette méthode est simple mais non sécurisée dans des applications réelles.

---

## Améliorations possibles
